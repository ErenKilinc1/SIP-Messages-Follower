# SIP-Messages-Follower
Bu araç, Cisco CUBE Gateway cihazlarına SSH üzerinden bağlanarak SIP mesajlarını gerçek zamanlı olarak takip etmenizi sağlar. Filtreleme yaparak, karmaşık debug logları arasından sadece sizinle ilgili olan çağrı trafiğini temiz ve okunabilir bir şekilde terminale basar. Özellikle transfer süreçlerini ve çağrı bacaklarını analiz etmek için idealdir.
